## Сайт для получения OAuth токена Яндекс Музыки

Сайт: https://ym-token.marshal.dev

Основной способ: вход через Яндекс (device flow). Пользователь подтверждает вход на странице Яндекса, токен создаётся прямо в браузере. Работает для всех аккаунтов.

Есть запасной вход по логину и паролю на старой версии авторизации Яндекса. Он работает не для всех аккаунтов.

Сайт также собирает все способы получения токена: расширение для браузера и приложение для Android.

### Стек

- [Vite](https://vite.dev/) + TypeScript, без фреймворков
- Менеджер пакетов: [pnpm](https://pnpm.io/)
- Двуязычный интерфейс (RU и EN), деплой на GitHub Pages

### Разработка

```bash
pnpm install
pnpm dev      # локальный сервер
pnpm build    # сборка в dist/
pnpm deploy   # публикация на GitHub Pages
```
